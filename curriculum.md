# 비개발 교육 운영자용 Claude Code/Codex 커리큘럼

## 교육 목표

교육 운영자가 Claude Code와 Codex를 코딩 도구가 아니라 업무 자료를 함께 보고 정리하는 AI 파트너로 이해하게 한다.

수강생은 교육 후 다음을 할 수 있어야 한다.

- ChatGPT/Gemini와 Claude Code/Codex의 차이를 업무 장면으로 설명한다.
- Claude Code와 Codex를 업무 목적에 맞게 구분한다.
- `CLAUDE.md`, `AGENTS.md`를 AI 업무 매뉴얼로 작성한다.
- MCP를 업무 자료 연결 통로로 이해한다.
- Clarify로 모호한 업무 요청을 실행 의뢰서로 바꾼다.
- 만족도 리포트와 기업별 할 일 목록을 만든다.
- 자신의 반복 업무를 Skill 초안으로 정리한다.

## 1일 진행 순서

### 1. ChatGPT/Gemini와 무엇이 다른가

ChatGPT/Gemini는 질문에 답하는 AI로, Claude Code/Codex는 내 폴더와 자료를 같이 보며 일하는 AI 비서로 설명한다.

### 2. 언제 Claude Code, 언제 Codex를 쓰나

Claude Code는 서비스 기획, 아이데이션, 운영 개선안, 사용자 여정 정리처럼 생각을 넓히고 다듬는 업무에 적합하다.

Codex는 이미 업무 절차나 원하는 결과물이 명확한 작업에 적합하다. 예를 들어 만족도 의견을 정해진 카테고리로 분류하기, 기업별 투두 리스트 만들기, 메일 초안을 정해진 톤으로 작성하기, 여러 파일을 읽어 정해진 양식의 리포트로 만들기, 반복 업무를 템플릿이나 Skill로 자동화하기에 적합하다.

### 3. AI 업무 매뉴얼 만들기

`CLAUDE.md`와 `AGENTS.md`를 업무 매뉴얼로 소개한다. 수강생은 개인정보 처리, 만족도 의견 분류 기준, 운영진 의견 말투, 메일 초안 검토 원칙을 문서로 남긴다.

### 4. MCP 이해하기: AI가 내 업무 자료를 찾는 통로

MCP를 Slack, Drive, Gmail, Notion 같은 업무 자료실로 이어지는 연결 통로로 설명한다.

실제 연결이 가능하면 연결된 도구를 보여준다. 연결이 어렵다면 Drive, Slack, Gmail에서 export한 파일을 `practice/private-data/`에 넣는 방식으로 대체한다.

이 단계의 목표는 AI가 내 자료를 같이 볼 수 있어야 업무를 맡길 수 있다는 점을 이해하는 것이다.

### 5. Clarify로 모호한 업무 요청 정리하기

“만족도 리포트 정리해줘”, “고객사별 할 일 뽑아줘”, “CS 답변 초안 만들어줘” 같은 요청을 Clarify로 구체화한다.

결과물은 업무 실행 의뢰서다.

- 목표
- 입력 자료
- 결과물 형태
- 분류 기준
- 말투
- 주의사항
- 개인정보 처리 기준

### 6. Lab 1: 만족도 조사 정성 의견 리포팅

Clarify로 정리한 요청서를 바탕으로 만족도 의견을 카테고리화하고, 카테고리별 개수, 대표 의견, 운영진 제안 문구, 강사용 리포트 요약을 만든다.

여기서 수강생은 좋은 요청서가 있으면 결과물 품질이 달라진다는 점을 체감한다.

### 7. Lab 2: 싱크업/메일 히스토리 기반 기업별 할 일 정리

싱크업 문서, 고객 메일, 운영 메모를 바탕으로 기업별 투두 리스트와 첫 회신 메일 초안을 만든다.

이 실습은 MCP와 자료 연결 개념을 실제 업무로 확인하는 단계다.

### 8. 나만의 업무 레시피 만들기

6번 또는 7번에서 했던 업무 중 하나를 골라 개인용 Skill 초안을 만든다.

Clarify 결과물을 그대로 스킬 설계 재료로 사용한다.

예시:

- 만족도 리포터
- 기업별 투두 정리기
- 수강생 안내 메일 작성기
- CS 답변 초안기

## Teaching Message

- MCP = AI가 자료를 찾으러 가는 길
- Clarify = AI에게 맡길 일을 정확히 적는 의뢰서
- Lab = 의뢰서를 바탕으로 실제 업무를 처리해보는 시간
- Skill = 잘 된 의뢰서를 매번 다시 쓰지 않도록 저장한 업무 레시피

## 최종 성취물

- 만족도 요약 1개
- 기업별 할 일 목록 1개
- 메일 초안 1개
- 개인 업무 레시피 1개
