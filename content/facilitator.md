# 강사용 진행 팁

이 과정은 코딩 교육이 아닙니다.
강사는 계속 교육 운영 업무 언어로 연결해줘야 합니다.

## 권장 진행

1. 처음에는 반드시 샘플 데이터로 시작합니다.
2. AI 업무 매뉴얼 단계에서 수강생 답변을 너무 오래 받지 않습니다.
3. MCP는 “자료 연결”이라고 먼저 설명합니다.
4. Clarify는 완성형 예시를 먼저 사용합니다.
5. 실데이터 적용은 마지막에 선택 과제로 둡니다.

## 막히는 지점

| 지점 | 대응 |
|---|---|
| GitHub 링크를 AI가 못 여는 경우 | ZIP 다운로드 방식으로 안내 |
| MCP 연결이 안 되는 경우 | 샘플 데이터 또는 export 파일로 진행 |
| Clarify 질문이 길어지는 경우 | 3개 이하 질문 원칙을 다시 안내 |
| 결과물이 너무 장황한 경우 | 표와 짧은 문단으로 다시 정리 요청 |

## 강사용 멘트

> 오늘은 샘플 데이터로 흐름을 익힙니다. 실제 자료는 나중에 `private-data`에 넣어 같은 방식으로 다시 실행하면 됩니다.
