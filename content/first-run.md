# 내 Skill 첫 실행하기

Clarify가 만든 `첫 실행 프롬프트`를 그대로 실행해보는 단계입니다.
여기서는 새로운 예제 프롬프트를 따로 만들지 않습니다.

## 진행 방식

1. Clarify 결과물에서 `첫 실행 프롬프트`를 찾습니다.
2. 연결된 내 실제 업무 자료가 맞는지 확인합니다.
3. 첫 실행 프롬프트를 Claude Code 또는 Codex에 붙여넣습니다.
4. 결과물을 보고 부족한 기준을 표시합니다.
5. 그 부족한 기준을 다음 단계에서 개인 업무 레시피에 반영합니다.

## 확인할 것

- 내가 원한 결과물 형태로 나왔는가?
- 필요한 자료를 제대로 사용했는가?
- 개인정보나 고객사 민감정보가 노출되지 않았는가?
- 다음에도 반복해서 쓸 수 있을 만큼 기준이 분명한가?

## 중요한 기준

첫 실행 결과는 완성본이 아니라 **내 Skill을 보강하기 위한 초안**입니다.
결과가 애매하면 실패가 아니라, 업무 레시피에 더 적어야 할 기준을 찾은 것입니다.
