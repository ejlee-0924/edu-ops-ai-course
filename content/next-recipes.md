# 다음 업무 레시피 후보

첫 번째 업무 레시피를 만든 뒤에는 팀에서 반복되는 다른 업무로 확장합니다.

## 추천 후보

| 업무 레시피 | 설명 |
|---|---|
| 수강생 안내 메일 작성기 | 차수/분반별 안내 메일 초안 작성 |
| 강사용 만족도 공유기 | 강사에게 전달할 요약만 따로 작성 |
| CS 답변 초안기 | 반복 문의에 대한 답변 초안 작성 |
| 기업별 투두 싱크 | 메일/싱크업/메모에서 할 일 정리 |
| 교육 종료 리포트 작성기 | 만족도, 출석, 개선사항을 한 문서로 정리 |

## 고르는 기준

- 매주 반복되는가?
- 기준이 비교적 명확한가?
- 결과물을 사람이 검토할 수 있는가?
- 개인정보 처리 기준을 정할 수 있는가?

이 네 가지가 맞으면 업무 레시피 후보로 좋습니다.
