# 환영합니다

오늘은 AI 도구를 배우는 시간이 아닙니다.
**내 반복 운영 업무를 AI에게 맡기는 시스템**을 만드는 시간입니다.

교육 운영자는 매번 비슷한 일을 반복합니다.

- 만족도 조사 정성 의견을 읽고 분류하기
- 강사님께 전달할 요약 만들기
- 고객사 메일과 싱크업 문서에서 할 일 찾기
- 수강생 안내 메일과 CS 답변 초안 작성하기

이 웹사이트는 그 일을 Claude Code와 Codex에서 재현 가능한 흐름으로 바꿉니다.

## 오늘의 큰 흐름

1. Clarify 설치 또는 사용 가능 여부를 확인한다.
2. 내 실제 반복 업무 하나를 고른다.
3. Clarify가 질문을 하나씩 던지며 업무를 구조화한다.
4. 필요한 Google Sheet와 자료를 연결한다.
5. 첫 결과물을 만들어본다.
6. 잘 된 흐름을 개인 업무 Skill로 저장한다.

> AI에게 그냥 시키는 것과, 업무 기준/자료/의뢰서를 갖춰 맡기는 것은 결과가 다릅니다.

## 시작 프롬프트

Claude Code 또는 Codex에서 아래 문장을 붙여넣으면 됩니다.

```text
아래 교육 페이지 흐름대로 내 반복 업무를 개인용 Skill로 만들고 싶어.
https://edu-ops-ai-course-vercel.vercel.app

먼저 Clarify 사용 가능 여부를 확인하고, 나에게 질문을 하나씩 해줘.
```

## 입력하면 어떻게 진행되나요?

1. AI가 Clarify 설치 또는 사용 가능 여부를 먼저 확인합니다.
2. 준비가 안 되어 있으면 Claude Code 또는 Codex에 맞는 설치 방법을 안내합니다.
3. 첫 질문으로 “어떤 반복 업무를 줄이고 싶나요?”를 묻습니다.
4. 답변을 들은 뒤 자료 위치, 결과물 형태, 말투, 개인정보 기준을 하나씩 묻습니다.
5. 답변을 모아 업무 실행 의뢰서, 자료 목록, 첫 실행 프롬프트, 개인용 Skill 초안을 만듭니다.
6. Google Sheets MCP가 연결되어 있으면 내 실제 시트로 첫 결과물을 만들어봅니다.
