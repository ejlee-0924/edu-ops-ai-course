# 만족도 리포트 만들기

설계 단계에서 만든 의뢰서를 바탕으로 만족도 정성 의견을 정리합니다.

## 사용하는 자료

- `practice/sample-data/satisfaction-comments.csv`
- 또는 `practice/private-data/`에 넣은 실제 만족도 파일

## 실습 프롬프트

```text
practice/prompts/03-satisfaction-reporting.md를 읽고, 샘플 만족도 조사 파일로 만족도 정성 의견 리포트를 만들어줘.
앞에서 만든 업무 실행 의뢰서 기준을 반영해줘.
```

## 만들어야 할 결과물

- 전체 요약
- 카테고리별 의견 개수
- 대표 의견
- 긍정 포인트
- 개선 포인트
- 운영진 제안 의견
- 강사님께 전달할 요약
- 고객사 공유 전 확인 필요 사항

## 예시

완성 예시는 `practice/examples/satisfaction-report-example.md`에서 확인할 수 있습니다.
