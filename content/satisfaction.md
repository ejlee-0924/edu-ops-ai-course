# 만족도 리포트 만들기

설계 단계에서 만든 의뢰서를 바탕으로 만족도 정성 의견을 정리합니다.

## 사용하는 자료

- Google Sheets MCP로 연결된 내 실제 만족도 조사 Google Sheet

## 실습 프롬프트

```text
Google Sheets MCP로 연결된 내 실제 만족도 조사 시트를 기준으로 만족도 정성 의견 리포트를 만들어줘.
앞에서 만든 업무 실행 의뢰서 기준을 반영해줘.

분류 기준:
- 콘텐츠/커리큘럼
- 강사/전달력
- 실습/참여
- 운영/안내
- 일정/환경
- 추가 요청
- 기타

결과물에는 다음을 포함해줘:
1. 전체 요약
2. 카테고리별 의견 개수
3. 카테고리별 대표 의견
4. 긍정 포인트
5. 개선 포인트
6. 운영진 제안 의견
7. 강사님께 전달할 요약
8. 고객사에 공유하기 전 확인 필요 사항

주의:
- 없는 의견은 만들지 마.
- 실명이나 민감정보는 제거해.
- 운영진 제안 의견은 정중하고 실행 가능한 말투로 써줘.
```

## 만들어야 할 결과물

- 전체 요약
- 카테고리별 의견 개수
- 대표 의견
- 긍정 포인트
- 개선 포인트
- 운영진 제안 의견
- 강사님께 전달할 요약
- 고객사 공유 전 확인 필요 사항
