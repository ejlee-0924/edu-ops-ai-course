# 내 업무 Skill로 시작하기

이 과정은 샘플을 따라 하는 방식보다, 수강생이 실제로 반복하는 업무 하나를 고르는 방식이 더 잘 맞습니다.

먼저 `설치하고 시작하기`에서 Clarify 설치 또는 사용 가능 여부를 확인합니다.
Clarify가 바로 설치되지 않아도 이 페이지의 시작 프롬프트로 같은 질문 흐름을 진행할 수 있습니다.

처음부터 완성된 요청을 쓰려고 하지 않아도 됩니다.
AI가 질문자가 되어 하나씩 묻고, 수강생은 자기 업무 기준으로 답합니다.
그 답변이 나중에 업무 실행 의뢰서와 개인 Skill 초안의 재료가 됩니다.

## 먼저 고르는 업무

아래 중 하나를 고르거나, 내 업무를 직접 적습니다.

- 만족도 정성 의견 리포트
- 고객사별 할 일 정리
- 수강생 안내 메일 작성
- CS 답변 초안 작성
- 교육 종료 후 결과 공유 자료 만들기
- 매주 운영 현황 요약

## Ask Your Question 방식

한 번에 긴 양식을 채우지 않습니다.
AI가 한 번에 하나씩 질문하고, 답변을 들은 뒤 다음 질문으로 넘어갑니다.

좋은 질문 흐름은 이렇습니다.

1. 어떤 반복 업무를 줄이고 싶은가요?
2. 그 업무에 쓰는 실제 자료는 어디에 있나요?
3. 최종 결과물은 어떤 형태여야 하나요?
4. 결과물을 누가 읽거나 사용하나요?
5. 말투나 표현에서 지켜야 할 기준은 무엇인가요?
6. 개인정보나 고객사 민감정보는 어떻게 처리해야 하나요?
7. 잘 된 결과물의 기준은 무엇인가요?

## 시작 프롬프트

```text
나는 비개발 교육 운영자야.
내 실제 반복 업무 하나를 AI에게 맡기는 개인용 Skill로 만들고 싶어.

Ask Your Question 방식으로 진행해줘.
한 번에 양식을 주지 말고, 질문을 하나씩만 해줘.
내가 답하면 그 답을 반영해서 다음 질문으로 넘어가줘.

최종적으로는 아래 결과물을 만들어줘:
1. 업무 실행 의뢰서
2. 필요한 업무 자료 목록
3. 첫 실행 프롬프트
4. 개인용 Skill 초안

먼저 내가 어떤 반복 업무를 줄이고 싶은지 질문해줘.
```

## 이 단계의 목표

이 단계가 끝나면 수강생은 “내 업무를 AI에게 맡기려면 무엇을 정해야 하는지”를 직접 경험합니다.
Clarify는 그 경험을 업무 실행 의뢰서로 정리하는 단계입니다.
