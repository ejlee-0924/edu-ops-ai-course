# 전체 학습 지도

이번 과정은 다음 흐름으로 진행됩니다.

```text
AI 업무 매뉴얼
→ Google Sheets 자료 연결
→ 업무 실행 의뢰서
→ 운영 업무 결과물
→ 개인 업무 레시피
```

## 왜 이 순서인가요?

AI에게 먼저 업무 기준을 알려주지 않으면 결과물의 말투와 기준이 흔들립니다.

Google Sheets에서 어떤 탭을 봐야 하는지 확인하지 않으면 AI가 추측할 가능성이 커집니다.

의뢰서 없이 바로 “정리해줘”라고 하면 결과물 형태가 매번 달라질 수 있습니다.

그래서 설계한 뒤 만듭니다.

## 챕터 연결

- `설계하기`: 기준과 구조를 만든다.
- `만들기`: 설계한 기준으로 실제 결과물을 만든다.
- `마무리`: 내 실제 자료로 바꿔 적용한다.
