# 전체 학습 지도

이번 과정은 다음 흐름으로 진행됩니다.

```text
Clarify 설치 확인
→ 내 반복 업무 선택
→ Clarify 질문
→ 업무 실행 의뢰서
→ 업무 자료 연결
→ 첫 결과물
→ 개인 업무 Skill
```

## 왜 이 순서인가요?

처음부터 샘플을 따라 하면 “내 업무에 어떻게 적용하지?”에서 멈추기 쉽습니다.

그래서 먼저 Clarify 사용 가능 여부를 확인하고, 내 반복 업무를 고른 뒤, Clarify가 질문을 하나씩 던지게 합니다.

그 답변이 업무 실행 의뢰서가 되고, 업무 자료 연결과 첫 결과물 제작 기준이 됩니다.

마지막에는 잘 된 흐름을 개인 Skill 초안으로 저장합니다.

## 챕터 연결

- `설계하기`: 내 업무를 질문으로 구조화한다.
- `만들기`: 구조화한 업무로 첫 결과물과 Skill 초안을 만든다.
- `마무리`: 내 실제 자료로 바꿔 적용한다.
