# 내 자료로 바로 적용하기

기본은 내 실제 업무 자료로 바로 시작하는 것입니다.

## 추천 순서

1. 실제 반복 업무에 쓰는 자료실을 준비합니다.
2. 개인정보를 제거하거나 익명화합니다.
3. MCP나 Connector에서 해당 자료를 연결합니다.
4. 설계하기 > Clarify로 내 업무 구조화하기부터 다시 진행합니다.
5. 결과물을 내가 실제로 쓰는 기준과 비교합니다.

## 주의

처음부터 모든 실제 자료를 넣지 마세요.
한 가지 자료만 바꿔도 흐름을 검증할 수 있습니다.

## 다시 실행할 프롬프트

```text
연결된 내 실제 업무 자료를 기준으로, 설계하기 > Clarify로 내 업무 구조화하기부터 다시 진행해줘.
Clarify 질문을 하나씩 받으면서 업무 실행 의뢰서와 개인용 Skill 초안까지 만들어줘.
개인정보와 고객사 민감정보가 결과물에 노출되지 않도록 먼저 확인해줘.
```
