# 내 자료로 바꿔보기

샘플 데이터로 한 번 해본 뒤, 실제 자료를 하나만 바꿔 넣습니다.

## 추천 순서

1. 실제 만족도 조사 파일을 준비합니다.
2. 개인정보를 제거합니다.
3. `practice/private-data/`에 넣습니다.
4. 설계하기 > 자료 연결 이해하기(MCP)부터 다시 진행합니다.
5. 결과물을 샘플 결과와 비교합니다.

## 주의

처음부터 모든 실제 자료를 넣지 마세요.
한 가지 자료만 바꿔도 흐름을 검증할 수 있습니다.

## 다시 실행할 프롬프트

```text
practice/private-data 폴더에 넣은 내 실제 만족도 조사 자료를 기준으로, 설계하기 > 자료 연결 이해하기(MCP)부터 다시 진행해줘.
개인정보와 고객사 민감정보가 결과물에 노출되지 않도록 먼저 확인해줘.
```
