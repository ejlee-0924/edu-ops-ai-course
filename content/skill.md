# 나만의 업무 레시피 보강하기

Skill은 내 반복 업무를 다시 맡길 수 있게 저장한 업무 레시피입니다.

Skill 초안은 앞의 Clarify 단계에서 이미 만들어집니다.
이 단계에서는 그 초안을 실제 첫 결과물 기준으로 보강합니다.

## 보강할 수 있는 레시피

- 만족도 리포터
- 기업별 투두 정리기
- 수강생 안내 메일 작성기
- CS 답변 초안기

## 실습 프롬프트

```text
Clarify에서 만든 개인용 Skill 초안을 보강해줘.

선택한 업무:
[내가 방금 구조화한 실제 업무명]

내 실제 업무 자료, 업무 실행 의뢰서, 결과물 기준을 반영해줘.
아직 첫 결과물이 없다면 보강 전 초안으로 두고, 첫 실행 후 보강할 항목을 따로 표시해줘.
개발자 용어보다 교육 운영자 업무 언어로 써줘.

아래 구조로 정리해줘:
1. 스킬 이름
2. 언제 쓰는지
3. 필요한 입력 자료
4. 처리 순서
5. 출력 양식
6. 주의사항
7. 좋은 결과물 기준
8. 개인용 Skill 초안
```

## 좋은 업무 레시피 기준

- 언제 쓰는지 분명하다.
- 필요한 입력 자료가 분명하다.
- 처리 순서가 단계별로 보인다.
- 출력 양식이 있다.
- 주의사항과 개인정보 처리 원칙이 있다.
