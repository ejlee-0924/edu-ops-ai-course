# 나만의 업무 레시피 만들기

Skill은 잘 된 업무 흐름을 저장한 반복 업무 레시피입니다.

오늘 만든 만족도 리포트나 기업별 할 일 정리 흐름 중 하나를 골라 Skill 초안으로 바꿉니다.

## 고를 수 있는 레시피

- 만족도 리포터
- 기업별 투두 정리기
- 수강생 안내 메일 작성기
- CS 답변 초안기

## 실습 프롬프트

```text
practice/prompts/05-skill-draft.md를 읽고, 방금 만든 만족도 리포트 흐름을 개인용 Skill 초안으로 바꿔줘.
개발자 용어보다 교육 운영자 업무 언어로 써줘.
```

## 좋은 업무 레시피 기준

- 언제 쓰는지 분명하다.
- 필요한 입력 자료가 분명하다.
- 처리 순서가 단계별로 보인다.
- 출력 양식이 있다.
- 주의사항과 개인정보 처리 원칙이 있다.
