## 진행 요청

수강생이 아래처럼 요청하면 됩니다.

```text
START_HERE.md와 README.md를 읽고, 교육 운영자 실습을 4번 MCP부터 순서대로 진행해줘.
나는 비개발자니까 개발 용어는 쉽게 풀어서 설명해줘.
```

## 진행 순서

1. `README.md`를 읽고 전체 흐름을 설명합니다.
2. `AGENTS.md`와 `CLAUDE.md`의 업무 매뉴얼을 확인합니다.
3. `sample-data/`와 `private-data/`의 자료 위치를 확인합니다.
4. `prompts/01-mcp-material-check.md`부터 순서대로 진행합니다.
5. 실습 결과는 수강생이 요청하면 `outputs/` 폴더에 저장합니다.

## 강사용 주의

이 실습은 코딩 교육이 아닙니다. 모든 설명은 교육 운영자 업무 언어로 진행합니다.
