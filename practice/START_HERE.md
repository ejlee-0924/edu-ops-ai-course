# START HERE

이 파일을 Claude Code 또는 Codex가 먼저 읽으면 됩니다.

## 진행 요청

수강생이 아래처럼 요청하면 됩니다.

```text
START_HERE.md와 README.md를 읽고, 설계하기 > 내 업무 Skill로 시작하기부터 진행해줘.
샘플보다 내 실제 반복 업무와 Google Sheet를 기준으로 진행하고, Clarify 질문을 하나씩 받으면서 구조화해줘.
나는 비개발자니까 개발 용어는 쉽게 풀어서 설명해줘.
```

## 진행 순서

1. `README.md`를 읽고 웹사이트 흐름을 설명합니다.
2. 수강생의 실제 반복 업무와 Google Sheet가 있는지 먼저 확인합니다.
3. `prompts/00-start-with-my-skill.md`로 Ask Your Question 방식 Clarify를 시작합니다.
4. 필요할 때 `AGENTS.md`와 `CLAUDE.md`의 업무 매뉴얼을 확인합니다.
5. Google Sheets MCP 샘플 워크북과 `sample-data/`, `private-data/`는 백업 자료로만 안내합니다.
6. 실습 결과는 수강생이 요청하면 `outputs/` 폴더에 저장합니다.

## 웹사이트 흐름

수강생에게는 번호보다 아래 여정으로 설명합니다.

```text
시작하기 → 이해하기 → 설계하기 → 만들기 → 마무리
```

실습은 `설계하기 > 내 업무 Skill로 시작하기`부터 시작합니다.

## 강사용 주의

이 실습은 코딩 교육이 아닙니다. 모든 설명은 교육 운영자 업무 언어로 진행합니다.
