# 05. 나만의 업무 레시피 보강하기

```text
Clarify에서 만든 개인용 Skill 초안을 보강해줘.

선택한 업무:
[내가 방금 구조화한 실제 업무명]

이 스킬은 비개발 교육 운영자가 반복 업무를 처리할 때 쓰는 업무 레시피야.

Clarify 결과물, 연결한 업무 자료, 방금 만든 첫 결과물을 참고해서 아래 구조로 정리해줘.
아직 첫 결과물이 없다면 보강 전 초안으로 두고, 첫 실행 후 보강할 항목을 따로 표시해줘:

1. 스킬 이름
2. 언제 쓰는지
3. 필요한 입력 자료
4. 처리 순서
5. 출력 양식
6. 주의사항
7. 좋은 결과물 기준
8. 개인용 Skill 초안

주의:
- 개발자 용어보다 교육 운영자 업무 언어를 써줘.
- 개인정보 처리와 검토용 초안 원칙을 포함해줘.
- 처음 보는 사람이 이 레시피만 보고 따라할 수 있게 써줘.
```
