# 05. 나만의 업무 레시피 만들기

```text
방금 수행한 업무 중 하나를 골라 개인용 Skill 초안으로 만들어줘.

선택한 업무:
[만족도 리포터 / 기업별 투두 정리기 / 수강생 안내 메일 작성기 / CS 답변 초안기]

이 스킬은 비개발 교육 운영자가 반복 업무를 처리할 때 쓰는 업무 레시피야.

Clarify 결과물과 방금 만든 결과물을 참고해서 아래 구조로 작성해줘:

1. 스킬 이름
2. 언제 쓰는지
3. 필요한 입력 자료
4. 처리 순서
5. 출력 양식
6. 주의사항
7. 좋은 결과물 기준
8. SKILL.md 초안

주의:
- 개발자 용어보다 교육 운영자 업무 언어를 써줘.
- 개인정보 처리와 검토용 초안 원칙을 포함해줘.
- 처음 보는 사람이 이 스킬만 읽고 따라할 수 있게 써줘.
```
