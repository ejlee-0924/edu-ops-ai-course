# 00. AI 업무 매뉴얼 만들기

아래 요청문을 Claude Code 또는 Codex에 붙여넣으세요.

```text
나는 비개발 교육 운영자야.
이 실습을 시작하기 전에 AI가 내 업무 기준을 따라 작업할 수 있도록 "AI 업무 매뉴얼"을 먼저 만들고 싶어.

먼저 나에게 3개 이하의 질문만 해서 내 업무 기준을 확인해줘.
질문은 꼭 필요한 것만 해줘.

질문 예시:
- 주로 다루는 교육 운영 업무는 무엇인가?
- 만족도 의견을 어떤 기준으로 분류하고 싶은가?
- 고객사 메일 초안은 어떤 말투여야 하는가?

내 답변을 바탕으로 다음 결과물을 만들어줘:
1. 내 업무 매뉴얼 1장
2. Claude Code/Codex에서 반복해서 쓸 업무 기준 문장
4. 개인정보/민감정보 처리 체크리스트
5. 다음 실습인 업무 자료 연결에서 확인할 자료 목록

주의:
- 개발자 용어는 쉽게 풀어서 설명해줘.
- 실제 고객사나 수강생 정보가 있으면 익명화 기준을 먼저 제안해줘.
- 메일 초안은 항상 "검토용 초안"이라고 표시하는 규칙을 포함해줘.
```
