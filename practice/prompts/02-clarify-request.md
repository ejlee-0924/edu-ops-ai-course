# 02. Clarify로 내 업무 구조화하기

```text
Clarify로 내 반복 업무를 구조화해줘.

나는 비개발 교육 운영자야.
내 실제 반복 업무 하나를 AI에게 맡기는 개인용 Skill로 만들고 싶어.

바로 결과물을 만들지 말고, 먼저 질문형 대화로 진행해줘.
필요한 질문을 한 번에 하나씩만 물어봐줘.
내가 답하면 그 답을 반영해서 다음 질문으로 넘어가줘.

질문이 끝나면 아래 결과물을 만들어줘:
1. 업무 실행 의뢰서
2. 필요한 업무 자료 목록
3. 첫 실행 프롬프트
4. 개인용 Skill 초안
5. 첫 실행 전에 확인해야 할 개인정보/민감정보 체크리스트

첫 질문은 "어떤 반복 업무를 줄이고 싶나요?"로 시작해줘.
```

자료가 아직 없는 수강생만 아래 완성형 예시를 사용합니다.

```text
다음 업무 요청을 Clarify 방식으로 정리해줘.

업무 요청:
"만족도 리포트 정리해줘"

나는 비개발 교육 운영자이고, 결과물은 실제 업무 검토용 초안으로 쓸 거야.
자료는 Google Sheets MCP로 연결된 Edu Ops AI Course Sample 문서의 satisfaction_comments 탭을 사용할 거야.

아래 항목을 반드시 포함한 업무 실행 의뢰서로 만들어줘:
- 목표
- 입력 자료
- 결과물 형태
- 분류 기준
- 원하는 말투
- 주의사항
- 개인정보 처리 기준
- 완료 기준

질문이 필요하면 3개 이하로만 물어보고,
합리적으로 가정할 수 있는 부분은 "가정"으로 표시해서 먼저 초안을 만들어줘.
```
