# 00. 내 업무 Skill로 시작하기

아래 요청문을 Claude Code 또는 Codex에 붙여넣으세요.

```text
나는 비개발 교육 운영자야.
내 실제 반복 업무 하나를 AI에게 맡기는 개인용 Skill로 만들고 싶어.

Ask Your Question 방식으로 진행해줘.
한 번에 양식을 주지 말고, 질문을 하나씩만 해줘.
내가 답하면 그 답을 반영해서 다음 질문으로 넘어가줘.

최종적으로는 아래 결과물을 만들어줘:
1. 업무 실행 의뢰서
2. 필요한 Google Sheet 또는 자료 목록
3. 첫 실행 프롬프트
4. 개인용 Skill 초안

먼저 내가 어떤 반복 업무를 줄이고 싶은지 질문해줘.
```
