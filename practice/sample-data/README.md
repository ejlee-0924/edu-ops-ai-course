# sample-data

샘플 자료는 실제 Google Sheet가 없거나 MCP 연결이 어려울 때 쓰는 백업 자료입니다.

## 기본 자료

| 자료 | 사용 방법 |
|---|---|
| `edu-ops-google-sheets-sample.xlsx` | Google Drive에 업로드한 뒤 Google Sheets로 열고, Google Sheets MCP에서 연결합니다. |
| `satisfaction-comments.csv` | Google Sheets MCP 연결이 안 될 때 쓰는 백업 파일입니다. |
| `syncup-notes.md` | Google Sheets MCP 연결이 안 될 때 쓰는 백업 파일입니다. |
| `client-email-thread.md` | Google Sheets MCP 연결이 안 될 때 쓰는 백업 파일입니다. |
| `cs-inquiries.md` | Google Sheets MCP 연결이 안 될 때 쓰는 백업 파일입니다. |

## Google Sheets 탭 구성

워크북을 Google Sheets로 열면 아래 탭을 사용합니다.

| 탭 | 내용 |
|---|---|
| `satisfaction_comments` | 만족도 조사 정성 의견 |
| `syncup_notes` | 싱크업 회의 메모 |
| `client_email_thread` | 고객사 메일 히스토리 |
| `cs_inquiries` | CS 문의 기록 |

실습의 기본 흐름은 수강생의 실제 Google Sheet를 Google Sheets MCP로 찾는 것입니다.
위 탭과 파일 기반 자료는 회사 보안, 계정, 네트워크 문제로 실제 자료 연결이 어려울 때만 사용합니다.
